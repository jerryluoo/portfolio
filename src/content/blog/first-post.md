Welcome to my blog! This is my first post on my new Astro-powered portfolio site.

I built this site using [Astro](https://astro.build), a modern static site generator that ships zero JavaScript by default. Combined with Tailwind CSS 4 and GitHub Pages hosting, it's a completely free, high-performance setup.

## Why Astro?

- **Zero JS by default** — pages load instantly
- **Islands architecture** — add interactivity only where needed
- **Content Collections** — type-safe Markdown blogging
- **View Transitions** — smooth page navigation built in
